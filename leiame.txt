Documentação para executar o projeto
1.	Criar um banco de dados Mysql com o nome “Selpe2”.
2.	Abrir o CMD e navegar até a pasta do projeto e rodar os seguintes comandos:

•	php artisan migrate
•	php artisan serve

3.	Realizar um registro na área de login.
4.	Efetuar Login.
5.	Pronto, agora basta usufruir da aplicação.
